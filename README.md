
# Chat Application

A simple real-time chat application built using Java + HTML.  
It allows multiple users to connect and chat with each other in real time over network (local / LAN / internet).

## features

- Real-time messaging between multiple clients.  
- Simple and clean user interface (HTML frontend).  
- Java backend for handling connections and message broadcasting.  
- Easy to run and test locally.
